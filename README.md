# 📊 Customer Churn Prediction

## 🚀 Overview

This project analyzes and predicts customer churn using machine learning techniques.

## 🧠 Model Used

* Logistic Regression (best performance)
* Optimized with custom threshold (0.3)
* Focus on maximizing Recall

## 📊 Results

* Recall: 0.96
* Precision: 0.41
* Accuracy: 0.62 (after tuning)
* Strong ability to detect churn customers

## 🔍 Key Steps

* Data Cleaning & Preprocessing
* Feature Engineering
* Model Training & Evaluation
* Threshold Optimization

## 🛠️ Technologies

* Python
* Pandas
* Scikit-learn


* Customers with high monthly charges and short tenure are more likely to churn
* Contract type significantly affects churn behavior

## 🎯 Goal

Help businesses identify at-risk customers and reduce churn.
